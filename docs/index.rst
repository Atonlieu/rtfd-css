rtfd-css
========

Custom ``*.css`` used in various projects. Usage:

#. Add `this`_ repository as a submodule
#. Modify ``conf.py`` (see `patch`_):
    * add ``rtdf-css`` directory to the static paths
    * load CSS by ``add_stylesheet``
